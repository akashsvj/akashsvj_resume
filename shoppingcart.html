EXTENUATING WEB VULNERABILITY WITH A DETECTION AND PROTECTION MECHANISM FOR A SECURE WEB ACCESS

Proposed system aims to find the vulnerability of a website. In a html content there is so many tags which can easy cause attacks. If that type of tags are more in developed site then their is a chance for web vulnerability. We have a dataset with tags which can leads to attack the site or not. We train a system using Naive Bayes algorithm and create a model. And when we are input a new url then it will check the vulnerability and predicts the output.
